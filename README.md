Support haptics and rumble (iOS14)

- Xbox One S/X Controller (not include XboxOne)
- Xbox Series Controller
- PS4 Controller
- PS5 Controller
- MagicKeyboard

#### Support External Display

- TypeC to HDMI
- Ligiting to HDMI
- Airplay

There are two purposes for this demo. 

- understanding and practicing the SpriteKit framework
- practicing the external-display and remote-controller
